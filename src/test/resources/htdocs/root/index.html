Welcome

The tamacat-httpd is a open source Java Web Server software, powered by " Apache HttpComponents".

This is a customizable HTTP/HTTPS Server framework and Reverse Proxy.

Features

  • Standards based, pure Java HTTP Server, implementation of HTTP versions 1.0 and 1.1
  • Pluggable architecture for custom request/response handler and filters
  • It provides handlers: Reverse Proxy, Tomcat Embedded, Thymeleaf Page and Static Contents Web Server
  • HTTPS multiple SSL certificates using Server Name Indication (SNI)
  • Tomcat Embedded Integration
  • Spring Boot Integration
  • Required Apache HttpComponents 4.5
  • Required Java Platform, Standard Edition 8 (JRE/JDK)
